FlexBox layout

The two axes of flexbox

  1. main axis
  2. cross axis

defined by flex-direction

  • row
  • row-reverse
  • column
  • column-reverse

Start and end lines

The flex items are placed within the container starting from start and going to end.

  1. main-start
  2. main-end
  3. cross-start
  4. cross-end

The flex container

An area of a document laid out using flexbox is called a flex container

Properties for the flex container :

  • display
  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items

The flex Item

The direct children of that container become flex items

Properties for the flex items :

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self
